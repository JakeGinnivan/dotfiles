# Jakes mac setup

### Specify the `$PATH`

```bash
export PATH="/usr/local/bin:$PATH"
```

### Copy .dotfiles

```bash
cp .aliases ~/
cp 
```


### Setup mac settings

```bash
./.macos
```

### Install software

```bash
./install.sh
```

### Git setup

```bash
git config --global user.name "<name>"
git config --global user.email "<email>"
```
